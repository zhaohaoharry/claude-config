Student Shiyu Zhang's master's thesis. Studies how social capital (home port identity) affects fishing vessel spatial choices in Chinese EEZ using Berry (1994) two-stage structural discrete choice model on GFW AIS data (2019). Key finding: same-port vessels avoid each other (lambda_s negative), interpreted as coordinated dispersion.

**Why:** Advisor review and guidance needed — student submitted all files as flat dump, code has significant identification and implementation issues.

**How to apply:** When working on this project, remember the key econometric vulnerabilities: (1) social_density endogeneity in clogit not addressed, (2) fleet centroid used instead of actual port coordinates, (3) Step 1 counterfactual code is commented out. These are the priority items for student guidance.

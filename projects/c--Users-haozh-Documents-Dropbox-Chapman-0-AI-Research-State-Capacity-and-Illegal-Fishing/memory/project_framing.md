Paper framing: "international order" — illegal fishing reveals that the post-WWII rules-based order fails where monitoring is weak and power is asymmetric. State capacity and resource curse are supporting literatures, not the primary frame.

**Why:** User explored several framings (pure state capacity, resource curse, international order) and chose international order as the most compelling angle for top-5 journals. The key empirical finding (stable states exploit weak states) maps naturally onto this narrative.

**How to apply:** When writing or revising, always connect findings back to the "rules-based order has boundaries drawn by surveillance" thesis. Don't let the paper drift into a pure fisheries/environmental paper.

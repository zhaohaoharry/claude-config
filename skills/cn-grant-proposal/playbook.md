# 中文经济学基金申请书写作手册
## 面向北京市社科基金 / 国家社科基金（应用经济·青年项目 / 一般项目）

> 本手册从 18 份中标申请书与申报经验文档中提炼，专为应用经济学方向、评审视角而写。每条给出“怎么做 + 为什么 + 示例措辞”。贯穿一句总纲：**评审时间有限，形式服务于让评审在三分钟内读懂你做什么、为什么值得做、你能不能做成。**

---

## 一、选题与标题

申请书的命运在标题与第一段就已决定一半。评审先看题目，再看摘要，最后才读正文——很多标书在这两关就被淘汰。

### 1.1 选题：以问题为导向，大题化小

- **选一个具体、明确的科学问题，而非领域或方向。** 把指南里宽泛的方向"化解"为自己能驾驭的小切口。为什么：口子小才利于深入，评审偏好"重点突出、利于深入"；范围越大越难判断你要解决什么。示例思路：不写"收入差距"，而锁定"收入差距的**代际传递**及其**变动趋势**"。
- **以问题为导向，不要以技术为导向。** 找到真问题课题就立住了；拿着新方法去找问题多半不成。
- **选未热的热点。** 评审滞后约半年，刚要热未热的方向是好题，纯热点上人扎堆、易撞车。例：2008 金融危机正发生时申报"美国金融危机成因与教训"。
- **选自己有积累、熟悉的领域。** 跨越太远难把自己"装得像行家"，前期成果也对不上。
- **三条自检：** 是不是一个真问题？重要性如何？我的思路是否对路？

### 1.2 标题：可拆解、有冲撞力、紧扣指南

- **标题至少改 10 遍，把它当作全文的"索引"。** 标题能拆出的几个关键词，应与综述每段、研究内容几点、创新几点**前后一一对应**。这是反复出现的中标共识。
- **用"主标题 + 副标题"双层结构。** 副标题保留/呼应指南条目名（保证被分到对的评审组），主标题点出自己的切口与新意。
  - 示例：*"信息化时代的新工业化与产业结构跨越式升级——工业化、信息化与跨越式发展研究"*
  - 示例：*"二元均衡视角下房地产过度金融化及其治理研究"*（视角 + 对象 + 落脚动词"治理"）
- **三段式句法常用：**`视角/方法 + 研究对象 + 落脚（机理/治理/对策/实证研究）`。
  - 例："基于 MRIO 模型的中国对外贸易隐含碳及排放责任研究"——先亮方法显技术含量，末尾"责任研究"把测算升格为制度议题。
- **嵌入一个差异化"钩子"概念**（自创或半自创术语），在全篇反复回扣，制造记忆点。如"链式迁移""结构性有效需求""过度金融化"。但**务必在正文明确定义**，否则被视为生造词。
- **显性挂靠指南条目。** 选题依据第一句即写明"本课题对应本年度指南'应用经济'部分第（XX）条"，先证明"对口"。
- **不要与指南条目字面完全相同**——既要被分对组，又要避免混同于同条目其他申请。
- **字数与主题词：** 课题名称一般 ≤40 汉字（含标点）；主题词只设 3 个，分层设计（一个定领域、两个亮视角/创新），与标题关键词完全对应。

> **经济学方向提示：** 标题落点优先放在具体的经济**关系/机制/弹性/效率/结构**上（如"代际收入弹性变动趋势""创新链与产业链融合机制"），而非笼统的"X 与 Y 研究"。

## 二、整体结构与各部分模板

严格复刻官方活页提纲，**标号与提纲一一吻合**，先抄录该栏要求、再作答，让评审"按图索骥"打分。篇幅按比例分配：**研究现状与意义 : 内容方法创新 : 前期成果与文献 ≈ 1 : 2 : 1**，把笔墨压在最能体现实力的"研究内容与创新"上。课题论证一般限 4000 字、约 4 页。

### 2.1 课题说明 / 摘要

- **最后写，但评审最先看。** 须讲清四块：现状、意义、构想、预期结果，且四块相互平衡，防止"头重脚轻"。
- **语气与正文相反——旗帜鲜明、不留弹性。** 正文论证走"中庸"、给不同意的专家留容忍空间；摘要则坚定自信。
- 字数受限时千锤百炼，杜绝空话套话。

### 2.2 选题依据（国内外研究现状述评 + 研究意义）

四层递进骨架：
1. **引入领域**（一两句话点背景与前景，可挂靠领导人原话/纲领文件定调）；
2. **学术史梳理**（列前沿代表人物与代表作，按子主题分块）；
3. **公允评说前人再转出不足**（"在充分尊重……基础上"，再用"但是"笔锋一转）；
4. **点明本课题价值与切入口**（用"鉴于此，本课题拟……"无缝接入）。

> 选题依据的总论先写一段无小标题的"背景→问题→意义"先声夺人，再用"（一）学术史梳理"分述。

### 2.3 研究内容

- **只列 3–5 个方面**（重大/一般项目可适当多，但切忌罗列过多、互相重叠）。由科学问题延伸出最本质相关的要素，重点突出、留足深挖空间。
- **每个研究内容模块用固定三件套：** ①一句话点核心任务 →②（1）（2）（3）列具体内容 →③一段讲机理逻辑。节奏统一、可预测，显严谨。
- **同时给两套结构**（中标范本常用法）：一套"主要内容"叙述版（讲逻辑），一套"总体框架/章节目录"版（秀完整体系）。
- **用漏斗式展开：** 研究对象 → 总体框架 → 重点难点 → 主要目标 → 思路方法。
- **重点与难点分开列**，且一一对应（重点=能做出的核心贡献，难点=最有挑战处 + 立即给出解决措施）。主动暴露难点反增可信度。
- 切忌写成"一本书/教科书的框架"——贪大求全失重点。

### 2.4 创新之处

- 见第四章（单列详述）。固定从 **3–4 条** 写，按维度命名（视角/理论、内容/思路、机制、方法）。

### 2.5 预期成果

- **形式明确、数量克制。** 列研究报告 + CSSCI 论文若干 + 资政/决策建议（应用经济强调对策落地）。例："研究报告 1 份、CSSCI 论文 3–5 篇、决策资政建议 2 份"。
- **阶段性成果别写太多**（2 项左右即可）——写满会让结项被动。
- **最终成果忌"论文集"**——首选专著或研究报告，论文集给人临时拼凑感。
- 预期结果须与科学问题、内容、目标首尾呼应（重质不重量）。

### 2.6 参考文献

- **求新、求权威：** 近期文献不可遗漏，最好引当年的权威期刊（《经济研究》《管理世界》等）。"有文献就等于没有疑问"。
- 中外文献交织，显示国际视野 + 本土熟悉度。
- 编号唯一、不重不漏——清单类内容务必逐条核对（中标范本常因编号重复而失分）。

### 2.7 研究基础 / 完成条件

- **三块组织：** 研究队伍 + 研究条件（数据/软件/平台）+ 学术思想/前期成果。
- **只列与本课题直接相关的成果**，无关条目一律不写——保持聚焦。
- 成果按重要性或刊物档次排列，最强最新放最前；可附社会评价（被引、转载、获奖、被部委采纳）作第三方背书。
- **活页匿名：** 前期成果只填成果形式、数量、刊物/出版社、独著或合著，**不得出现成果名称、作者姓名、单位、发表时间**；负责人姓名不得出现。
- 团队配置忌"挟大人物以自重"、忌写忙人/领导/过多在读学生。

## 三、语言与修辞（评审视角的可读性工程）

### 3.1 主题句前置（最高优先级）

- **怎么做：** 每段、每个编号点的第一句即给结论或判断，后文再展开证据。
- **为什么：** 评审快速扫读，读首句即抓要点。
- **示例：** *"相比金融化研究的系统和深入，学界对房地产金融化的关注不够。"*（先下断语，再举证）

### 3.2 加粗与分层

- 用多级编号制造条理：`一、（一）1.（1）①`。长文档读起来像提纲，降低阅读成本。
- **创新点/闪光句用黑体突出，但严控用量**（建议加粗总量 ≤500 字），否则满篇加粗等于没有重点。
- 每段约 6 行：太少说不清，太多评审看着累。

### 3.3 句式与节奏

- **长短句交错：** 背景与机理用长复句堆逻辑，下结论用短句斩钉截铁（"美国经济日趋金融化。"）。
- **整散结合、善用排比对仗**制造气势与记忆点：
  - *"工业化创造供给，城镇化创造需求"*
  - *"科技'强链'、市场'带链'、政策'融链'、开放'紧链'"*
- 写完**朗读三四遍**，检查停顿、语气、转折是否流畅铿锵。

### 3.4 正式书面语体

- 第三人称、判断句为主，几乎不用"我认为""我们觉得"；自指用"本课题""本研究""拟采用"。
- 多用确定性措辞："核心是""关键在于""必然要求""据此""综上"——建立权威。但**不要满篇"必然/无疑"**，过密显武断，需与具体证据搭配。

### 3.5 段落衔接

- 靠**显式连接词与编号系统**推进，而非自然过渡：因此 / 综上 / 其中 / 反过来 / 值得关注的是 / 鉴于此。
- 每个文献子块结尾用一句承上启下（"仍然缺乏……""还需将……相结合"），自然滑向研究空白。

### 3.6 显得权威可信

- **机制语言**而非关系语言：不写"有关系"，写"通过四种渠道""背后的逻辑涉及……"，显示懂深层机理。
- **数字替代形容词：** 不写"成果丰富"，写"发表论文 22 篇、被引最多 45 次、2 份建议被部委采纳"。
- **精确引证：** 每个判断后跟"（作者，年份）"，密集而精准；政策原文加书名号 + 确切日期。

### 3.7 避免 AI 腔与口语化（经济学申请书专项）

- **删掉万能空话：** "随着……的不断发展""在当今社会""众所周知""具有重要的现实意义和理论价值"（泛说无据）。要换成**具体的数字、机制、政策抓手**。
- **不要机械三连排比堆砌**（"……的……、……的……、……的……"无差别罗列）——每一项要承载不同信息，否则显空泛。
- **慎用形象比喻的密度：** "融化剂/催化剂/助推器/粘合剂"连用过密会沦为口号，每个比喻后必须跟实质论证。
- 不用网络化、口语化表达；不用第一人称感叹；不用"我们将会发现"式悬念句。
- 不滥用"虽然……但是……"句式（引出空白的逻辑可借鉴，但措辞要变换，避免通篇雷同显模板化）。
- **概念先界定后使用**——自造术语必须给定义，否则评审视为生造词而非创新。

## 四、创新点写法

- **必须有一个明确的"新"：** 新方法老问题 / 老方法新问题 / 新方法新问题，三者居其一即可，但**必须有**。
- **3–4 条、高度抽象、宁精勿多。** 多而散评审记不住卖点；中标范本常把核心卖点浓缩成一句"最主要创新之处是……"。
- **分维度命名、对号入座：** 学术思想（视角）/ 学术观点（命题）/ 研究方法，恰好对应活页三类创新要求。
- **创新与研究空白严格镜像、形成闭环：** 综述指出几个"缺乏"，创新就逐一回应。"空白—填补"对位，让创新可验证而非自夸。
- **创新写成可检验的命题**，而非口号。
  - 例："拟提出'结构性有效需求'理论，将经济周期理论与增长理论统一起来。"
  - 例（命题式）："当地方政府土地经营收益持续大于房地产税预计收入时，改革不能成功。"
- **用未来式动词框定为可交付贡献：** "将构建……""拟提出……""将区分……为……"。
- **方法创新 = 具体方法清单 + 用途**，不空喊"更先进"。指明它纠正了哪个具体偏差、带来什么可验证能力。
  - 例："用双样本工具变量法克服单年收入替代造成的向下偏误，为分位数回归与城乡分维提供基础。"
- **红线：** 绝不轻率宣称"填补空白/国内独创/国际领先"——落选项目约 1/5 栽在此。人文社科真正的"创新"有限，**确保有鲜明"特色"比硬吹创新更稳妥**。

## 五、学术 / 应用 / 政治价值与国家战略契合

- **分栏写学术价值与应用价值**，各 2–3 条编号，每条用"做了什么 → 带来什么"句式。
  - 学术："构建……理论模型，拓展……理论基础。"
- **应用价值落到可操作政策抓手**（应用经济的命门）：列出具体政策领域清单（"土地、财政、教育、就业、医疗、养老、住房保障"），用密度显示落地广度；并用三问收口："做什么（存在问题）？如何做（对策）？会做得如何（预期效果）？"
- **政治价值嵌入措辞，而非单列口号：** 用"高质量发展""新型举国体制""双循环""房住不炒""共同但有区别的责任""防范系统性风险"等政策热词对齐国家战略；引最新中央/部委原文（中央经济工作会议、五年规划、国务院意见）加书名号 + 日期。
- **把研究对象的重大性转化为研究的重要性**：用大数字制造紧迫感（"未来将有 2–3 亿农民进入城市"）。
- **价值要兼顾基础与实用，分寸要稳：** "不能过分强调，也不能过于平淡"。意义忌过宏观、忌庞杂（有落选项目仅意义一项写 700 字却看不出究竟有何价值）。
- **国际对标抬高紧迫性：** "避免重蹈拉美城市贫民窟覆辙""抢占新一轮国际竞争制高点"。
- **设"相对于同类项目的新进展"小节**：数出近 5 年同类立项（带编号），再讲本课题几点边际贡献，证明不重复。

## 六、文献综述与研究空白

综述是衡量学术水平的关键，是"说服评审的支点"——"不能把握前人成果，何谈超越"。

- **按子主题（不按时间）分块组织**，每块内再用"一是/二是/三是"梳理流派；制造"全景已览、唯此一隙未补"的印象。
  - 经济学常见分块：理论谱系（如熊彼特→新增长→新制度）/ 子领域（创新链理论 / 两链关系 / 数字化影响）/ 单边-双边-全球。
- **达到三层要求：** ①熟悉已做与在做的工作 →②做出**判断**（方法是否科学、理论是否深入、哪些已解决、哪些未解决、有无尚未被提出的新问题）→③定位自己的新起点与突破点。
- **先扬后抑、公允评说：** 给前人"相当高度评价"后再"笔锋一转"用"但是"引出不足。"肯定别人就是肯定自己。"
- **空白要具体到可被同行识别的技术细节**，而非泛说"研究不足"：
  - 例："多数模型是静态的、不含时间变量""用单年收入作替代变量产生向下偏误""技术同质性假设将影响结论准确性"。
- **空白与研究内容严丝合缝（综述挖坑、内容填坑）：** 收尾用"综上，已有研究颇丰，但仍有以下欠缺：（1）（2）（3）……针对以上问题，本课题拟……"。
- **本土化空白**是应用经济的有力理由："大多基于西方发达国家经验，借鉴意义有限，适合我国国情的系统分析仍然缺乏。"
- **重要人物与代表作必须具名点出**；近期/当年权威文献不可遗漏。
- **败笔清单（务必避免）：** 几乎无述评；只介绍同行皆知的名词概念；只罗列书目不谈观点差异；以自己观点替代述评；述评偏激贬低前人（招致评审逆反）。

## 七、研究方案 / 技术路线 / 可行性

- **思路用一条主线串全文：** "提出问题—理论分析—实证分析—政策设计"，并配通俗注解"是什么—为什么—现状如何—怎么做"。五段研究框架与主线一一对应、逻辑闭环。
- **方法按用途分类、与研究内容逐一咬合**——"干什么用什么方法"。切忌罗列一堆高级方法却不说解决哪个问题；**少用自己不熟悉的方法**。
- **实证方法写到具体技术名 + 基本思想**（应用经济的硬实力信号）：
  - 例："双样本工具变量法（通过两个独立样本合成父子配对数据）→ 分位数回归（比 OLS 更稳健）→ 路径分解机制"。
  - 常见可点名工具：DSGE、VAR / 协整 / 格兰杰因果、断点回归、空间计量、三阶段 DEA、面板联立方程、ARDL、贝叶斯 MCMC、熵值法/层次分析法确定权重。
- **可行性重心放在"方案的思路"上**（反常识要点），而非人员设备的堆砌；但数据与团队仍要落实。
- **数据写成核心竞争力：** 交代来源、覆盖、跨期、独有优势、口径协调与简化假设。
  - 例："CHIP 数据（社科院 + 国家统计局）覆盖 10 多省、近 20 年四轮，是唯一包含家庭财产的跨期数据。"
- **主动披露简化假设并论证其无害**——以透明换信任；缺数据给可操作替代方案。
- **难点—措施成对出现：** 每个难点后紧跟"（1）（2）（3）"解决路径，把"难"转化为"有准备的难"。
- **技术路线配流程图/框图**，文字反复"（详见图 X）"回指，图文互证，让评审"看得见"。
- **研究计划给分阶段时间表**（时间段 + 任务 + 产出），把抽象计划落成可考核节点。一般 2–3 年，应用对策研究约 1 年。
- **可行性证据用可核查数字背书：** 已主持课题数、专著论文数、被引/转载/获奖、数据库与软件清单（Stata/EViews/Matlab/CSMAR/CHIP）、实验室与馆藏。
- **建议选题已实做一部分再申报**，研究内容才写得具体可信。

## 八、常见失败点与红线清单

**内容/逻辑类**
- 研究内容写成一本书/教科书框架，贪大求全、内容互相重叠。
- 综述只罗列不评价、找不到新起点；或贬低前人致评审逆反。
- 创新点贪多求全（罗列五六条空泛"创新"稀释可信度）或只有研究思路而无观点。
- 方法与内容脱节、方法堆砌、生拉硬扯套用时髦理论。
- 价值表述空喊口号、过宏观、与政策脱节；意义篇幅失衡。

**红线（直接被否）**
- 宣称"填补空白/国内独创/国际领先/国内一流"——高频落选病。
- 前期成果造假；拿未答辩或已通过的博士论文充当申报课题。
- 活页泄露个人/单位/姓名信息（违反匿名评审，取消资格）。
- 程序性自伤：人员超项、同室撞车、忽视回避制度、份数/装订错误。

**形式类（拉低专业印象）**
- 错别字、标点、格式错误——"第一印象决定 80% 判断"，务必逐字校对。
- 超字数（论证 ≤4000 字、标题 ≤40 字、主题词 ≤3）。
- 加粗过多失重点；段落过长；字体过小（小四号为宜）。
- 经费概算不合规、随意填写；参考文献编号重复错乱。

**期望管理类**
- 阶段性成果列太多致结项被动；最终成果写成"论文集"。
- 前期成果写得过满，让评审觉得课题已基本完成、无需资助。
- 自我感觉良好未必好——写完仍觉有一两个大问题没解决，反说明做得深入。

## 九、可直接套用的句式模板与金句库

**标题/选题**
1. `[视角/方法]视角下[研究对象]的[机制/治理/演化]研究`
2. `[主标题：自创概念点切口]——[副标题：呼应指南条目名]研究`
3. 选题依据首句："本课题对应本年度《课题指南》'应用经济'部分第（XX）条，紧扣……国家战略需要。"

**综述与空白**
4. "在充分尊重前人学术建树的基础上，[领域]研究已取得丰硕成果……**但是**，仍存在以下欠缺：（1）……（2）……（3）……"
5. "综上，已有相关研究颇丰，但仍有以下欠缺；**针对以上问题，本课题拟**……"
6. "适合我国国情的[X]系统分析仍然缺乏，现有研究多基于西方发达国家经验，借鉴意义有限。"
7. "已有文献形成两种研究范式……为我们提供了有益参考，**其不足之处是**……"

**创新与价值**
8. "本课题最主要的创新之处是[一句话方法/视角增量]。"
9. "拟提出并证明：[可检验的命题，机制 + 条件 + 结论]。"
10. "在保持较高学术层次的同时，突出战略研究的应用性与对策性。"
11. "本研究既有助于[理论贡献]，又能针对不同[群体/区域]提出可操作的政策改进措施。"
12. "做什么（存在问题）—如何做（措施对策）—会做得如何（预期效果）。"

**方法与可行性**
13. "研究遵循'提出问题—理论分析—实证分析—政策设计'主线，即'是什么—为什么—现状如何—怎么做'。"
14. "除一般熟知的研究方法外，本研究具体采用[方法清单]，以保证研究的科学性与先进性。"
15. "本步试图实现的研究目标是……（每一步方法后紧跟其目标）。"
16. "[数据集]覆盖……跨期……，是唯一包含[独有变量]的数据，可运用[软件]实现。"

**经验金句（写作心法，非正文用语）**
17. "形式大于内容——题目一看就有冲撞力，能吸引评委的眼球。"
18. "标题至少改 10 遍：它能拆出的子问题，应与综述、内容、创新前后一一对应。"
19. "有文献就等于没有疑问；可行性分析是你说服评委的第二次机会。"
20. "大题小做一般做不好；选题最好以问题为导向，不要以技术为导向。"
21. "肯定别人就是肯定自己；把握不了前人成果，怎么谈得上超越。"

## 十、提交前自检清单（25 条）

**选题与标题**
- [ ] 1. 标题已改 ≥10 遍，其关键词能与综述每段、研究内容各点、创新各点一一对应。
- [ ] 2. 标题 ≤40 字；主题词恰 3 个、与标题对应、含视角/创新预告。
- [ ] 3. 选题落到一个具体科学问题，而非领域/方向；切口小、利于深入。
- [ ] 4. 选题依据首句显性挂靠指南条目，且标题不与条目字面雷同。

**结构与篇幅**
- [ ] 5. 严格按官方活页提纲分段，标号一一吻合，先抄要求再作答。
- [ ] 6. 篇幅比例 ≈ 现状意义1 : 内容创新2 : 成果文献1；论证 ≤4000 字、约 4 页。
- [ ] 7. 摘要四块齐全（现状/意义/构想/预期）、平衡、语气坚定。
- [ ] 8. 研究内容 3–5 块，重点突出，未写成一本书的框架；重点与难点分列。

**综述与创新**
- [ ] 9. 综述按子主题分块，重要人物与代表作具名，含判断而非罗列。
- [ ] 10. 研究空白具体到技术细节，并与研究内容严丝合缝。
- [ ] 11. 全文无"填补空白/国内独创/国际领先"等自夸词，无贬低前人。
- [ ] 12. 创新 3–4 条、分维度命名、写成可检验命题、与空白镜像对位。

**方法与可行性**
- [ ] 13. 每个研究内容绑定具体方法并注明用途；无方法堆砌、无不熟悉的方法。
- [ ] 14. 实证方法点名具体技术 + 基本思想；数据来源/覆盖/独有优势写明。
- [ ] 15. 每个难点后紧跟解决措施；配技术路线图与分阶段时间表。
- [ ] 16. 简化假设已主动披露并论证无害。

**价值与措辞**
- [ ] 17. 学术与应用价值分栏，每条"做了什么→带来什么"；应用价值落到具体政策抓手。
- [ ] 18. 政治价值嵌入措辞、对齐国家战略，引文加书名号 + 日期；意义不庞杂、不过宏观。
- [ ] 19. 每段主题句前置；连接词显式衔接；机制语言而非关系语言。
- [ ] 20. 加粗总量 ≤500 字且只标创新/闪光句；无 AI 腔、无口语化、自造概念已定义。

**成果、合规、形式**
- [ ] 21. 预期成果形式明确、阶段性成果 ≤2 项、最终成果非论文集。
- [ ] 22. 研究基础只列相关成果、含第三方社会评价；团队无忙人/无挟大人物。
- [ ] 23. 活页完全匿名（无姓名/单位/成果名称/发表时间/负责人名）。
- [ ] 24. 参考文献求新、编号唯一不重不漏；经费概算合规。
- [ ] 25. 全文朗读 ≥3 遍、零错别字、字体小四、装订与份数正确；写完仍能指出一两个待深化处。
